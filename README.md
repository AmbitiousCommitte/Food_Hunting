# Food_Hunting
This repo is all about food hunting. It operates a discord Bot to daily update info about free food events at USF. It also provides a web page to display the food events.

## How to use the bot
1. Clone the repo
2. Install the required packages
```bash
pip install -r requirements.txt
```

3. Run the bot
```bash
python bot.py
```

## Feature
1. Give info about free food events at 12am everyday
2. Response to user's command '!food' to give the food events from current time to the end of the day
